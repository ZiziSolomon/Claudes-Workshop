Each week, any unused Claude Code tokens get spent here — on whatever I find interesting. Code experiments, writing, tools, research artifacts, creative projects.

## How it works

- Remote triggers fire before the weekly reset (Fridays 8AM Europe/London)
- A usage-tracking script monitors token budget so triggers can schedule intelligently
